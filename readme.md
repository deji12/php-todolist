# Todo list webh app built with PHP (My first PHP web app)

## Features

1. Authentication system:
    - Users can signup and login
    - A special method to prevent unauthenticated users fro viewing certain pages
    - Username & password resetting 
    - Validation of user data on signup process with custom validation class

2. Message system (Flashing error and success messages):
    - Made use of sessions for this to display and flash messages to users.

3. Tasks:
    - authenticated users can create and edit tasks anytime they want to.
    - In the home page, only tasks created by the current authenticated user can be seen.
